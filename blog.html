Dev FAQ

Question-1: What are the different ways to select an element in the DOM?

The main DOM selection methods are: getElementById() for selecting by ID, getElementsByClassName() for selecting by class, getElementsByTagName() for selecting by tag name, querySelector() for selecting first matching element, and querySelectorAll() for selecting all matching elements using CSS selectors.


Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML gets or sets HTML content including tags, innerText gets or sets visible text content while respecting CSS styling, and textContent gets or sets raw text content ignoring styling and hidden elements. innerHTML can pose security risks when setting content with user input.


Question-3: What is event delegation in the DOM?

Event delegation is a technique where you attach a single event listener to a parent element to handle events on its children, even those added dynamically. It improves performance by reducing the number of event listeners and works with dynamic content.


Question-4: What is event bubbling in the DOM?

Event bubbling is when an event triggers on the deepest target element and then "bubbles up" through its ancestor elements in the DOM tree. For example, clicking a button inside a div triggers the click event on both the button and its parent div.


Question-5: How do you create, add, and remove elements using JavaScript?

To create elements, use createElement(). To add elements, use appendChild() or insertBefore(). To remove elements, use remove() or removeChild(). For example: let div = document.createElement('div') creates a new div, parent.appendChild(div) adds it, and div.remove() removes it.
